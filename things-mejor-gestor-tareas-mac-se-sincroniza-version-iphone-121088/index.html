    Things, el mejor gestor de tareas para Mac, ya se sincroniza con la versión de iPhone

    22 agosto 2008 #mac#tareas#things

    La última actualización de Things para Mac ya permite sincronizar nuestras listas de To-Dos y proyectos entre las versiones de escritorio y de iPhone/iPod Touch.

    Things es una de las mejores alternativas para la gestión de tareas si sueles trabajar con un Mac o si dispones de un iPhone o un iPod Touch. Está basando en el método Getting Things Done, de David Allen. Una interfaz sencilla y clara, y el acceso rápido a las funciones más habituales lo convierten en el líder de las aplicaciones de su categoría, y la posibilidad de sincronización entre dispositivos ha acabado de completar el pack.

    La aplicación para iPhone tiene un precio de 7,99€ y la versión de escritorio de momento se ofrece de forma gratuíta, aunque anuncian que costará $49 cuando lancen la versión 1.0.
