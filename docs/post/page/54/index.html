Editor de texto para tu CMS

Uno de los principales objetivos de un CMS (un gestor de contenidos) es permitir que los usuarios puedan mantener un website sin necesidad de tener conocimientos técnicos ni de programación. La solución a esto pasa por varias opciones: Posibilitar la entrada sólamente de texto plano, y dar posteriormente un formato uniforme a ese texto, eliminando cualquier etiqueta HTML.
