Fundamentos Web 2006 en imágenes

Oct 8, 2006 #accesibilidad #congresos #desarrollo web #estandares #fotos #fundamentos web #fundamentosweb2006 #oviedo #usabilidad #web semántica

Aquí os dejo algunas de las fotografías que saqué durante el congreso Fundamentos Web 2006 , celebrado en Oviedo durante los días 3, 4 y 5 de octubre. Un saludo y mi agradecimiento desde aquí por la cálida acogida de los compañeros ovetenses.
