Things, el mejor gestor de tareas para Mac, ya se sincroniza con la versión de iPhone

La última actualización de Things para Mac ya permite sincronizar nuestras listas de To-Dos y proyectos entre las versiones de escritorio y de iPhone/iPod Touch. Things es una de las mejores alternativas para la gestión de tareas si sueles trabajar con un Mac o si dispones de un iPhone o un iPod Touch. Está basando en el método Getting Things Done, de David Allen. Una interfaz sencilla y clara, y el acceso rápido a las funciones más habituales lo convierten en el líder de las aplicaciones de su categoría, y la posibilidad de sincronización entre dispositivos ha acabado de completar el pack. »

Por fin llegaron las tarjetas de MOO cards

MOO.com es un servicio de impresión online que te permite diseñar tus propias tarjetas de visita y pegatinas, a partir de fotografías o imágenes hechas por ti o desde diferentes diseños que ellos mismos ofrecen. Además, se integra con el servicio de alojamiento de fotografías Flickr, así que si tienes fotos alojadas allí, resulta sencillísimo preparar un pack de tarjetas a partir de tus mejores fotos. Hace cosa de un par de meses realicé un pedido, animado al ver el fantàstico acabado de las tarjetas que acababa de pedir Albert. »
