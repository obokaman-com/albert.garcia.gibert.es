Editor de texto para tu CMS

Jan 17, 2005 #desarrollo web

Uno de los principales objetivos de un CMS (un gestor de contenidos) es permitir que los usuarios puedan mantener un website sin necesidad de tener conocimientos técnicos ni de programación. La solución a esto pasa por varias opciones:

  • Posibilitar la entrada sólamente de texto plano, y dar posteriormente un formato uniforme a ese texto, eliminando cualquier etiqueta HTML.
  • Permitir la entrada directa de algunas etiquetas sencillas de HTML.
  • Usar etiquetas propias para algunos formateos sencillos (negritas, cursivas, alineaciones de párrafo...)
  • Usar un editor de texto enriquecido WYSIWYG (algo tipo Word, FrontPage, Dreamweaver...), que permita modificar la apariencia del texto, y escriba el HTML necesario automáticamente.

Está claro que la opción preferible, la más completa y sencilla para el usuario final es la última, ya que las otras, de una manera u otra, o limitan la capacidad de incidir en la apariencia del contenido, o requieren algún tipo de aprendizaje del sistemas de etiquetas. El problema es que, hasta ahora, era bastante complicado, por no decir imposible, encontrar un editor avanzado de texto orientado a web que funcionara correctamente tanto en Internet Explorer como en Mozilla/Firefox, Opera... Lo dicho: hasta ahora.

El proyecto se llama HTMLArea 3.0, y actualmente se encuentra en fase BETA (creo que ya han publicado la RC 1). Es un editor muy completo que nos permitirá editar visualmente el contenido de textareas, generando XHTML 1.0/1.1 válido. Podemos personalizar las hojas de estilo que utilizarán los textareas, personalizar las barras de herramientas, activar opciones como menús contextuales y, como no, editar directamente el código XHTML.

Desde hace un par de días ya se encuentra integrado en el CMS de l blog colectivo de obokaman.com y la verdad es que funciona a las mil maravillas, tanto desde IE como desde Firefox. ¿No has participado nunca en el blog colectivo? Esta puede ser la ocasión. Únete al blog rellenando el sencillo formulario de registro. Podrás enviar textos e imágenes que aparecerán en la página principal de obokaman.com, y dispondrás de tu propio blog en www.obokaman.com/blog/tunombre ¿Quieres saber más?

Ahora tienes un motivo más para participar. ;)
