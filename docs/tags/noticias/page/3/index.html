Maquillando noticias

En este caso, sin embargo, no consigo entender el motivo que llevó  a este periódico a publicar dicho titular: "Guapo bailaor acusado de homicidio tras atropello". ¿!Se puede saber qué leches aporta en este caso que Farruquito sea o no guapo?!
